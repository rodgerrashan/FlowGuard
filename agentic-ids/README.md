# Agentic IDS

## Overview
Intelligent Intrusion Detection System powered by agentic AI capabilities.

## Features
- Real-time threat detection
- Autonomous response mechanisms
- Machine learning-based anomaly detection
- Integration with existing security infrastructure

## Installation
```bash
pip install -r requirements.txt
```

## Usage
To run IDS system
```python
python app.py
```

To start packet simulation
```python
cd simulation
python stream.py
```
